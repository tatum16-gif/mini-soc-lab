# Mini SOC Lab

A simulated Security Operations Center (SOC) monitoring tool built with C# and SQLite.

## Features
- Real-time log monitoring
- Brute force attack detection
- SQL injection detection
- Automated IP blocking simulation
- SQLite database logging
- Security alert generation

## Technologies Used
- C#
- .NET
- SQLite
- FileSystem Monitoring
- SQL Queries

## Example Alerts
 ALERT: Brute force attack from 192.168.1.99  
 ACTION: Blocking IP 192.168.1.99

## Purpose
This project demonstrates foundational cybersecurity concepts including:
- Threat detection
- Security monitoring
- Log analysis
- Incident response
- Security automation
